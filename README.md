## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Docker img with Alpine +
- install Node
```bash
$ npm install -g @nestjs/cli
```

## Build
- pull repository
```bash
$ npm install
$ npm run build
```
- start container process with $npm run start:prod

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
